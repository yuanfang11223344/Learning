# ChatGPT对话索引

> 最近更新时间：2026-06-12  
> 维护范围：ChatGPT 对话整理后的分类 Markdown  
> 规则：只保留整理后的分类笔记，不建立原始导出目录；Chrome 下载目录中的导出文件处理后删除。

## 归档规则

- 同类型内容放入同一个大类文件夹。
- 每个大类文件夹下按细分主题维护多个 Markdown 文件。
- 新内容进入前，先检索本索引和已有分类文件；相似、同义、同主题内容直接合并并丰富已有文件。
- 只有确实属于新主题时，才在对应大类文件夹中新建 Markdown 文件。
- 每个 Markdown 内部必须保留原问题和原回答，并补充摘要、分类、关键词、延伸解释、可执行建议和关联笔记。
- 新增、移动、合并、重命名文件或大类文件夹后，必须同步更新本索引。
- 本地整理目录、Obsidian 目录、Git 仓库目录三处结构保持一致。
