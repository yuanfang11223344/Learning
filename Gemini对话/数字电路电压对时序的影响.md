# 数字电路电压对时序的影响

> 来源：Gemini 对话，2026-05-26

---

## 一、电压与频率的协同评估

### Q: 怎么评估 0.75V 1.34G 和 0.8V 1.6G 的时序逻辑？

**A:** 电压提升和频率提升对时序的影响是**反方向**的。

| 变化 | 影响 | 数值 |
|------|------|------|
| 电压 0.75→0.8V (+7%) | 门延迟减小 | 约 10%~15% 改善 |
| 频率 1.34→1.6G (+19%) | 时钟周期缩短 | 746ps → 625ps（-16%） |

净效果 = 延迟改善（电压红利） vs 周期缩短（频率压力）。7% 的电压提升带来的延迟改善通常不足以抵消 16% 的周期缩短——**很可能 setup slack 变差。**

## 二、电压如何影响门延迟

### Q: 电压上升为什么会让时序变好？

**A:** MOS 管导通电流与过驱动电压的关系：

```
I_on ∝ (V_dd - V_th)²
```

Vdd↑ → 过驱动电压 (Vdd−Vth)↑ → 对负载电容的充放电电流增大 → 门延迟减小：

```
Vdd↑ → (Vdd-Vth)↑ → Ion↑ → t_pd↓ → 速度↑
```

**但代价是功耗**：`P = α·C·V²·f`，电压以平方项影响动态功耗。

## 三、PVT Corner 与 Vt 选型的联动

电压只是 PVT（Process/Voltage/Temperature）三维度之一。从 Vt 选型角度看：

| Corner | 电压 | 难点 | Vt 策略 |
|--------|------|------|---------|
| SS（Slow） | 低 | Setup 最难收敛 | 关键路径多用 LVT |
| FF（Fast） | 高 | Hold 最难收敛 | 控制 LVT 比例，防止 hold 恶化 |
| TT（Typical） | 标称 | 基准参考 | SVT 为主 |

> 电压在这里是 STA 分析条件，你无法控制它——你只能通过选择不同 Vt 的 cell 来适应给定的 PVT corner。

## 四、U_DLY 仿真延迟宏

### Q: `#U_DLY` 是干嘛的？

**A:** `` `U_DLY `` 是仿真专用的单位延迟宏：

```verilog
`define U_DLY #1      // 定义 1 个时间单位的 clk-to-q 延迟

always @(posedge clk) begin
    q <= `U_DLY d;    // 等效于 q <= #1 d;
end
```

**两个作用：**
1. 模拟真实器件的 clk-to-q 延迟，避免仿真中所有寄存器零延迟同时翻转造成的信号竞争
2. 让仿真波形更接近实际硬件行为（时钟沿后数据有微小延迟才变化）

### Q: 综合工具对 `<= #U_DLY` 报错 Intra-assignment delay？

**A:** `#delay` 是纯仿真行为，不能综合成硬件。解决方案：

```verilog
// 方案一：综合时用宏开关消除
`ifdef SYNTHESIS
    `define U_DLY          // 空定义，综合时 # 消失
`else
    `define U_DLY #1       // 仿真时保留延迟
`endif

// 方案二：用条件编译包围
always @(posedge clk) begin
`ifndef SYNTHESIS
    q <= `U_DLY d;
`else
    q <= d;
`endif
end
```

大多数现代流程中，Lint 工具报这个警告可以 waive 掉——只要确认宏在综合环境中正确定义为空。

## 五、总结与举一反三

### 核心要点

1. **Vdd↑ → 速度快 + 功耗高**——电压和时间是物理上不可兼得的。
2. **PVT 是 STA 的输入条件**——你的调节手段是 Vt 选型，不是电压本身。
3. **`#U_DLY` 是仿真专用，综合时必须消除**——宏开关是最干净的方案。

### 举一反三

- **Temperature Inversion（温度反转）**：先进工艺（FinFET 以下）中，低温不一定快——某些 corner 下温度上升反而加速（由于阈值电压温度系数变化）。这是 STA signoff 的新挑战。
- **Dynamic Voltage Scaling（DVS）**：芯片运行时动态调整电压——高频时升压、低频时降压，平衡性能和功耗。实现 DVS 需要 Level Shifter 和电源管理单元。
- **IR Drop 与电压**：芯片内部由于电源网络电阻，实际到达 cell 的 Vdd 低于标称值——瞬间大电流区域的 IR Drop 可能吃掉电压红利，严重时导致 setup 违例。
